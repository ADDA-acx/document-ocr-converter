# Changelog

## 1.0.0 - 2026-06-12

- Added top-level switching between PDF-to-Word and image-to-Excel workflows.
- Added bordered-table detection with OpenCV.
- Added OCR-coordinate fallback for borderless tables.
- Added formatted `.xlsx` output with batch processing.
- Added a Windows 7 SP1 x64 one-file build.
- Reorganized the project into a standard `src` layout.
- Added bilingual documentation, tests, CI, and open-source project metadata.
